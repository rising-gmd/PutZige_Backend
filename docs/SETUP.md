# Development Setup

## Prerequisites
- .NET SDK (matching the solution requirements)
- IDE or editor with C# support

## Initial setup
1. Clone the repository
2. Restore packages with `dotnet restore`
3. Build the solution with `dotnet build`

## Running the API
- `dotnet run --project PutZige.API/PutZige.API.csproj`

## Recommendations
- Enable nullable reference types in new code
- Run tests before committing changes
